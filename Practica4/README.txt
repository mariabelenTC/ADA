El programa cumple todas las funcionalidades que se especifican en el enunciado.

Únicamente da problemas cuando, usando la estructura de arrays para el map, superas el máximo de usuarios permitidos del chat.
También cuando habiendo superado el máximo de usuarios (usando lista dinámica), intentas meter un usuario repetido (cuando metes usuarios repetidos antes de superar el máximo, el programa funciona perfectamente).
